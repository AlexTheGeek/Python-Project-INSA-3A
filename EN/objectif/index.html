Objectives

Python is a very popular programming language in the IT world for carrying out various tasks such as automation, websites, Artificial Intelligence (AI), etc. Our objective is to understand the given code in detail but also to simplify it, keeping only the dependencies useful for the project. This will enable us to produce clear documentation to use the script on a new computer and by a person not necessarily trained for its use. In a second step, we will be able to extend this code to the databases available on the kaggle site.
